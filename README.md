# 🌱 AgriGrow – Smart Farming Website

AgriGrow is a modern and responsive farming website developed using HTML, CSS, and JavaScript.  
The project focuses on smart agriculture, sustainable farming solutions, and modern web design practices.

---

## 🚀 Features

- Responsive Multi-Page Design
- Modern Navigation Bar with Dropdown Menu
- Attractive Hero Section
- Farming Services Showcase
- About Us Section
- Contact Page
- Clean and Professional UI
- Mobile Friendly Layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts

---

## 📱 Responsive Design

The website is optimized for:

- Desktop Devices
- Tablets
- Mobile Phones
